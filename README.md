# python-employee-management-system

## Overview

This project is a console-based employee management system developed using Python and object-oriented programming principles.

The application demonstrates class-based design, inheritance, and employee information management by modeling different employee types within an organization.

## Features

* Employee management system
* Object-oriented programming (OOP)
* Class inheritance
* Employee information tracking
* Office and field employee structures
* Console-based interaction

## OOP Concepts Implemented

### Classes

* Employee
* Office Employee
* Field Employee

### Object-Oriented Principles

* Inheritance
* Encapsulation
* Method Definition
* Class-Based Design

## Technologies Used

* Python
* Object-Oriented Programming (OOP)

## Skills Demonstrated

* Python Programming
* OOP Design
* Class Creation
* Inheritance
* Employee Data Management
* Console Application Development

## Project Purpose

The main purpose of this project is to practice object-oriented programming concepts and demonstrate how inheritance can be used to model different employee roles in a business environment.
